Tools Developed

capable

2022 A tool for RNA Cap type prediction in RNA Nanopore sequencing data

Capable is a python package that uses machine learning to predict RNA cap structures in Nanopore RNA sequencing data

https://github.com/adnaniazi/capable

tailfindr

2019 Poly(A) tail length estimator for Oxford Nanopore RNA and DNA reads

tailfindr is an R package to profile poly(A) tail lengths in Oxford Nanopore's RNA and cDNA sequencing data

https://github.com/adnaniazi/tailfindr

Analyze4D

2012 fMRI Visualization and Voxel Time-course Analysis Tool

This MATLAB®-based tool can be used to intuitively visualize and do region-of-interest and voxel time-course analysis of fMRI and real-time fMRI data.

www.analyze4d.com

Donders Real-time fMRI

2011 A General Purpose Pipeline for Brain Decoding and Neurofeedback Studies

Developed a real-time functional magnetic resonance imagining pipeline that can retrieve, preprocess, and decode fMRI brain scans with machine learning in real time for neurofeedback studies.

http://tiny.cc/rtfmri
